I-BOT

I-Bot (Information Bot) is an artificial-intelligent-based chatbot that provides you with information (about University of Uyo, Main Campus) in real time as responses to queries by the user. It uses a simple user interface to collect user's queries or question and provides appropriate responses with the ability to learn from users' queries and match similar queries.

What?

Provides answers specific to entity-based queries (that contain the 'WHAT' phrase).

Where?

Provides answers to location-based queries (that contain the 'WHERE' phrase).

When?

Provides answers to time-based queries (that contain the 'WHEN' phrase).

How?

Provides answers to process-based queries (that contain the 'HOW' phrase).

Who?

Provides answers to person-based queries (that contain the 'WHO' phrase).
